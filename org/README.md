# 共享任务目录

新建任务：复制 `tasks/TASK-TEMPLATE.md` 为 `TASK-<YYYYMMDD>-<slug>.md`

建议字段：

- `owner`（枢衡为 `main`） `phase` `phase_transitions` `last_activity_iso` `status`
- `assignees` `cost_notes` `capability_gap`（绳墨） `sibo_status` `halt`

止损与自动推进见 **`AUTOPILOT.md`**；一键停止：`scripts/stop.sh`；备份：`scripts/backup.sh`。
